# 🚀 Problem Solving: Dr. Mostafa Saad's Sheet

A structured repository containing my solutions to the legendary "Dr. Mostafa Saad" Competitive Programming sheet. This repo tracks my journey from Junior Phase A to Advanced algorithmic mastery.

---

## 🛠️ Tech Stack & Standards

- **Language:** C++20
- **Compiler:** GCC (MinGW-w64)
- **Core Focus:** Standard Template Library (STL), Time/Space Complexity Optimization, and Clean Code.
- **Boilerplate:** Optimized for Fast I/O using `ios_base::sync_with_stdio(false); cin.tie(NULL);`.

---

## 📂 Project Structure

Each phase is organized to maintain a clean workspace and easy navigation.

```text
.
├── Phase-A/           # Junior Training (Basics, Greedy, Brute Force)
├── Phase-B/           # Intermediate (Data Structures, Recursion, Math)
├── Phase-C/           # Advanced (Graph Theory, Dynamic Programming)
├── templates/         # Reusable C++ Boilerplate
└── .gitignore         # Ignores binaries and IDE configs
```
